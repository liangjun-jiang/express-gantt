What's Gantt Plot

A Gantt chart is a type of bar chart that illustrates a project schedule. ... Gantt charts illustrate the start and finish dates of the terminal elements and summary elements of a project. Terminal elements and summary elements constitute the work breakdown structure of the project. See [wiki for the detail]

Why Gantt Plot for Project Plannning

Quoted from [ Purpose of a Gantt Chart ], a Gantt Chart is a timeline that is used as a project management tool to illustrate how the project will run. You can view individual tasks, their durations and the sequencing of these tasks. View the overall timeline of the project and the expected completion date.
